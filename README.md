# EM-LLM: Electromagnetics Expert
A fine-tuned Mistral-7B model specialized in Electromagnetics using LoRA (Low-Rank Adaptation).

## Project Overview
- **Base Model:** Mistral-7B-Instruct-v0.2
- **Knowledge Base:** Sadiku's Elements of Electromagnetics (PDF)
- **Technique:** QLoRA fine-tuning (1 epoch)
- **Infrastructure:** Google Colab (Training) & Hugging Face (Deployment)
